# unity-fbx-textures-export
Unity Editor tool that exports FBX models with all textures in one click. Automatically finds materials, organizes files, and creates Blender-ready packages. Saves hours of manual work.  Unity modellerini texture'larıyla tek tıkla FBX olarak aktaran araç. Material'leri otomatik bulur, dosyaları organize eder. Blender için hazır paket oluşturur.
